About HelpHub

HelpHub was inspired by the need to connect communities with essential local services. During a challenging time in my community, I realized how difficult it was for people to find resources like food banks, shelters, and healthcare. This project aims to bridge that gap and make it easier for individuals to access the help they need.

This project is a Portfolio Project for Holberton School. You can find more about Holberton School here.
